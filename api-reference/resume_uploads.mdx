## Overview
The One Click Apply API requires a public resume link for processing, if you have a local PDF resume file, you can use our `/upload_resume` API to upload it and get a public URL.

This public URL can then be used with our [/apply](https://docs.oneclickapply.co/api-reference/endpoint/apply) API to apply for a job.

You need to authorize the call with your API Key.

## Example Implementation
- Here's a simple [JS implementation](https://replit.com/@Marvy101/OCA-uploadresume-JS) in [Replit](https://replit.com).

<RequestExample>

```bash cURL
curl --request POST \
--url https://oneclickapply.co/api/v1/upload_resume \
--header 'API-Key: YOUR_API_KEY' \
--form 'file=@/path/to/your/Resume.pdf'
```

```python Python
import requests

url = "https://oneclickapply.co/api/v1/upload_resume"
api_key = "<api-key>"
file_path = "/path/to/your/Resume.pdf"
headers = {
    'API-Key': api_key
}
files = {
    'file': open(file_path, 'rb')
}
response = requests.post(url, headers=headers, files=files)

print(response.json())
```
</RequestExample>
